## 1. Design Principles

### 1.1 Core Principles
1. **Hebrew First (RTL)** - Entire interface right-to-left
2. **Touch Friendly** - Large targets for kitchen environment
3. **Offline Aware** - Always show connection/sync status
4. **Speed** - Minimize taps to complete common tasks
5. **Clarity** - Clear visual hierarchy, readable in busy kitchen

### 1.2 Design Constraints
- Must work on mobile phones (320px+)
- Must work with wet/dirty hands (large touch areas)
- Must be readable in various lighting conditions
- Must print clearly on standard A4 printer
